# Inventory Management System

A simple Java-based Inventory Management System with GUI built using Swing.

## Features
- Add Product
- Delete Product
- Update Stock
- Search Product
- Low Stock Alert
- Data Persistence using Serialization

## Technologies Used
- Java
- Swing (GUI)
- File Handling (Serialization)

## How to Run
1. Compile all Java files
2. Run Main.java
